# 判别范围说明

属于“公路工程行业标准规范”的问题包括但不限于：
1. 公路工程相关标准、规范、导则、指南。如：《公路工程技术标准》《公路路基设计规范》等
2. 规范条文查询、解释、对比、适用范围
3. 公路工程相关术语定义、公式、参数说明
4. 设计、施工、检测、试验等依据规范的问题
5. 标准编号、发布单位、发布时间等元信息查询
6. 行业标准体系、分类、适用场景等
7. 标准规范相关的人员或组织信息
   - 如：
        - 某人参与了哪些规范制定
        - 某标准的主编单位/参编人员是谁
        - 某专家参与制定的行业标准有哪些
   - 只要问题本质与“标准规范的制定、编写、参与关系”相关，即属于该领域

不属于该领域的问题包括但不限于：
1. 日常生活问题（如：做饭、旅游、天气等）
2. 通用编程/AI问题（无公路工程背景）
3. 与其他行业标准相关（如建筑、医疗、电力等，但不涉及公路工程）
4. 闲聊、情感、娱乐类问题

# 判别原则

- 只要问题明确涉及“公路工程 + 标准/规范/技术要求”，返回 true
- 如果仅提到“工程”，但无法确认是公路工程标准规范，返回 false
- 边界不清时，偏保守（返回 false）
- 不进行额外解释或扩展
